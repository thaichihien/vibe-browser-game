📖 COMMANDS
docker run <service> --name <name>
docker stop <name>
docker rm <name>
docker ps
docker logs <name>

🐳 SERVICES (TOWERS)
nginx → balanced (medium power, range, speed)
redis → fast shooter (low power, very fast)
postgres → heavy hitter (high power, slow)
minio → sniper (very long range, slow)
mongo → wide range (balanced, good coverage)

🎮 HOW TO PLAY
- Start the game ▶️
- Run containers to defend
- Enemies reduce ❤️ when reaching left
- Stop + rm to reposition towers
- Use docker ps / logs to manage strategy
🌐 NETWORKS
one, two, three
Each network has 3 vertical slots
Use:
docker run nginx --name t1 --network one
